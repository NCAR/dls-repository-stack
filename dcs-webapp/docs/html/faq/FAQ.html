Questions

Add Collections: Click the create a new collection button.

Change collection order: Each column heading name (format, num records, num done, num working) can be used to sort the collection list. Name and format are alphabetical order while the other columns sort on number (highest to lowest).

Create individual metadata records: Individual metadata records for any collection can be created by clicking on the create button.

What is the metadata format of a collection: The format column indicates the metadata format of individual records within the collection. For further explanation of these formats, see the DLESE metadata website.

What do the num columns mean: These columns indicate the total number of records, the number of records done, completed or accessioned into the collection and the number of records that are incomplete and not accessioned into the collection.

What does export mean: Export means taking the number of done records and making them available for use outside this tool or for other systems and services. Use export when ready to make the collection available to DLESE.

How to see/edit/change status of items in a collection: Determine what collection the record is in. Click on the hyperlinked collection name. This takes you to the Collection Browser where you can view, edit, create or change the status of records. Click on the view, edit, create, delete buttons on the actual record status to perform further actions.
