Tool Help - Fetch by ID

The ADN editor is able to fetch records from the DLESE library by record ID. To fetch a record from DLESE, enter a valid ID in the Record ID field and select the Fetch by ID button.

To try this feature out, use this ID: DLESE-000-000-004-409

Note: Records fetched from DLESE are saved locally when the write button is selected.

Implementation Details

The Fetch by ID feature uses DDS web services to retrieve records.
