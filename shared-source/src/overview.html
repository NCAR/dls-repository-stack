DLS Java Libraries (dls-libs) is collection of Java packages used in the Digital Discovery System (DDS), the Digital Catalog System (DCS), and other Java applications. DLS EL Function Library and other applications.

This software is developed by Digital Learning Sciences (DLS) (http://dls.ucar.edu/) at the University Corporation for Atmospheric Research (http://www.ucar.edu/).

See the DLS Expression Language Function Library descriptions. These functions are for use in JSP pages and JSTL tags.
